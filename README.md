# Angular Coding Test

## Calculator Specification
Create an angular application which is able to calculate only addition or subtraction of numbers.

Front-end 
* one simple page with form, put for example (1+3) or (30-9), submit a button and show result.

Back-end
* choice any web server running on Node.js (your favourite, Express, etc.)
* create REST API, call calculating logic
* implement calculating logic


Not required:
* no beautiful css style (simple HTML), focus on clean code.
* don't create tests


Required
* Angular 11
* production ready code
* well documented
* send a github link with project
* documented how to run the application
